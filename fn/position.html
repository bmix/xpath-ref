fn:position

Returns the context position from the dynamic context.

Signature

fn:position() as xs:integer

Properties

This function is deterministic, context-dependent, and focus-dependent.

Rules

Returns the context position from the dynamic context. (See Dynamic Context Components.)

Error Conditions

A dynamic error is raised [ERRXPDY0002] if the context item is absent.
